### 发布/部署
    假设自己的网页全部开发完毕
    a.网址（通过这个网址直接访问自己的网页） 腾讯云/阿里云 购买域名
    b.服务器（把我的网页放在这个服务器上 供外部访问） 腾讯云/阿里云 购买域名

    屌丝的做法：白嫖
    github的服务器和自动生成的网址
    1. www.github.com 注册一个账号 并登录（不稳定）
    2. 创建一个仓库 reposi （项目文件夹）
       new ->  填写一个仓库名称（必须是 昵称.github.io）
       add README 打开
    3. 将自己的网页文件上传到这个仓库中
       进入这个仓库 -> ADD file -> Upload files -> 将网页文件拖拽到指定区域 自动上传
       在使用commit changes 下方写上自己刚才做了什么（日志）
    4. 使用仓库名在网址中打开自己的网页 昵称.github.com.io
    5.为这个网址生成一个对应的二维码（多乐塔二维码在线生成器）
      网址链接 -> 输入完整的网址 -> 生成二维码
